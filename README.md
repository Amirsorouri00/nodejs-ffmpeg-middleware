# nodejs-ffmpeg-middleware


## Installation

### Back-End
````
$ git clone <repository_url>
$ cd to the cloned repo
$ npm install 
$ node index.js
Server is Up Now...
````

### Client-Side
use demo.jwplayer in Usefull Links and make a request to the url below:
* http://127.0.0.1:8000/streams/stream.m3u8
